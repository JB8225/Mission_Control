## ⚡ KEY INSIGHT: WHY 35% IS BETTER THAN 60%

```
┌─────────────────────────────────────────────────────────────────────┐
│                    OLD vs. NEW STRUCTURE                            │
├────────────────────────────┬────────────────────────────────────────┤
│  OLD STRUCTURE (60%)       │  NEW STRUCTURE (35%)                  │
├────────────────────────────┼────────────────────────────────────────┤
│  Investor Pool: 60%        │  Investor Pool: 35%                   │
│  Shares: 12 × $50,000      │  Shares: 7 × $50,000                  │
│  Capital Raised: $600,000  │  Capital Raised: $350,000             │
│  Per Share Ownership: 5%   │  Per Share Ownership: 5%              │
│  JB Equity: ~5–10%         │  JB Equity: 20%                       │
│  Mgmt/Team Equity: ~15%    │  Mgmt/Team Equity: 27%                │
│  Woody Rollover: ~15%      │  Woody Rollover: 18%                  │
└────────────────────────────┴────────────────────────────────────────┘
```

**The per-share investor economics are IDENTICAL** — each share still represents 5%
of the company. What changes is everything else:

1. **Less capital to raise** — $350K vs. $600K. Easier to close, faster to fund.
2. **JB retains meaningful equity** — 20% creates strong operator alignment.
3. **Management is properly incentivized** — 27% pool drives performance.
4. **Simpler cap table** — 7 investors vs. 12. Fewer votes, cleaner governance.
5. **Same investor upside** — each share earns identical distributions and exit proceeds.

**Bottom line:** The revised structure is better for the deal AND better for investors.
Fewer shares outstanding = faster raise = higher probability of close.
